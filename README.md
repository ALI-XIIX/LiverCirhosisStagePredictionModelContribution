# Liver Cirrhosis Stage Prediction – Exploratory Data Analysis

## Project Overview
This project was part of a **team data science project** aimed at analyzing and modeling a Liver Cirrhosis Stage Prediction dataset. The overall goal of the project was to follow a structured data science workflow, beginning with exploratory data analysis (EDA) and progressing toward predictive modeling of disease stages.

Liver cirrhosis is a progressive liver disease that can be categorized into different stages of severity. Understanding how clinical and demographic features relate to disease progression can help identify patterns that may assist in predicting patient outcomes.

## Objective
The primary objectives of the project were to:

- Explore and understand the structure of the dataset
- Analyze relationships between patient features and cirrhosis stages
- Identify patterns in both categorical and numerical variables
- Prepare the dataset for machine learning models that could predict cirrhosis stage

## My Contribution
My contribution to the project focused on **exploratory data analysis and visualization**. The analysis included:

- Generating statistical summaries of the dataset
- Identifying numerical and categorical features
- Performing **univariate analysis** of categorical variables using count plots
- Performing **bivariate analysis** comparing variables with the cirrhosis stage
- Visualizing distributions of numerical features using histograms
- Investigating relationships between important medical variables

This analysis helped reveal patterns in the data and supported the later modeling stage of the project.

## Key Visual Analyses
Several relationships between clinical variables were explored using visualizations, including:

- Albumin vs Bilirubin
- Bilirubin vs Prothrombin
- Platelets vs Prothrombin
- SGOT vs Bilirubin
- Cholesterol vs Triglycerides
- Copper vs SGOT
- Alkaline Phosphatase vs Bilirubin

These visualizations were used to identify possible correlations between medical indicators and the stage of liver cirrhosis.
